Verse Hunt

A Bible memory challenge where you race the clock to find a random scripture reference and type the first three words of the verse.

Created by

Easton Berg

Verse Hunt was built to make scripture memory fun — encouraging players to open their Bible, recall God's Word, and grow more familiar with the text.
